# Hyerix Homebrew tap

Homebrew formulae for Hyerix's open-source tools.

## Install

```sh
brew install hyerix/tap/hyerix-mcp
```

That single command auto-taps `hyerix/tap` and installs the formula in one go. If you'd rather tap explicitly first:

```sh
brew tap hyerix/tap
brew install hyerix-mcp
```

## Formulae

| Formula | Source repo | Description |
|---------|-------------|-------------|
| `hyerix-mcp` | [hyerix/hyerix-mcp](https://github.com/hyerix/hyerix-mcp) | stdio MCP server for NATS — 14 read-safe tools for AI agents. |
